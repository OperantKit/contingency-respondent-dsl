# Tier B プリミティブ — インデックス

> プリミティブ毎に 1 ページ。本インデックスは 26 個の Tier B パヴロフ
> 手続きを概念的族ごとにグループ化し、各プリミティブの DSL 綴りと
> 一次出典を示す。

26 個のプリミティブはすべて Core の `ExtensionRespondentPrimitive`
拡張ポイント (`contingency-dsl/spec/en/respondent/grammar.md §4`) を
通じて登録される。どれも Tier A プリミティブを再定義しない。
